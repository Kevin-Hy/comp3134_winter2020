1) What is the definition of the server port?

The answer is it is a communication endpoint. Physical and wireless connections are 
terminated at ports of hardware devices, and are identified by 16-bit unsigned numbers.

2) What is server port range?

The answer is 0 to 65535

3) How many bits (the number in binary [2^N]) does number above represent?

The answer is 16

4) State 10 common server ports

The answer is 20,21,22,23,25,53,80,110,119,123
